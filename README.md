# Graph Calculator

A small program designed to draw mathematical graphs from expressions. Suitable for students, teachers, and anyone who needs quick visualization of math functions.

## Features

- Draw mathematical graphs from expressions
- Zoom in/out for better visualization
- Support for keyboard and mouse input

## Requirements

- Built with Visual Studio
- .NET Framework


## Language

- C#


## Installation

Clone the project and open it with Visual Studio:

```bash
git clone https://github.com/your-username/graph-calculator.git
cd graph-calculator
```

Then, build and run the project in Visual Studio.
